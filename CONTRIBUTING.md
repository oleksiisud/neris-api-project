# Contributing to NERIS API Project

Thank you for considering contributing to the NERIS API Project! We welcome contributions from everyone. Please follow these guidelines to help us maintain a welcoming and productive environment.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** with clear, descriptive commit messages.
3. **Test your changes** to ensure they work as expected.
4. **Open a Pull Request** with a clear description of your changes and the motivation behind them.

## Code of Conduct

Please be respectful and considerate in all interactions. Harassment or inappropriate behavior will not be tolerated.

## Reporting Issues

If you find a bug or have a feature request, please open an issue with detailed information.
